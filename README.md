# 🎁 电子礼簿系统

**✨ 双击即用，无需安装，无需服务器！**

## ✨ 核心特性

### 🖥️ 副屏展示（投影专用）

- **实时同步**：主屏录入，2秒内自动更新
- **大字体显示**：标题和数据字体放大，远距离清晰可见
- **单行展示**：显示最新的12条记录，无空白区域
- **最新高亮**：最后一条记录脉冲动画高亮
- **高度自适应**：完美填充屏幕，视觉舒适
- **主题适配**：喜事红色 / 丧事灰色

### 📄 打印存档

- **横屏A4**：充分利用纸张空间
- **全量打印**：一次性打印所有礼金记录
- **彩色主题**：喜事红色 / 丧事灰色自动适配
- **竖排格式**：传统礼簿风格，符合中式习惯
- **自动统计**：包含总金额、人数、支付方式统计

### 💾 数据导入导出（Excel统一方案）

- **📊 导出数据**：导出当前事件为Excel（礼金明细 + 统计汇总）
- **📚 导出全部**：导出所有事件为Excel（多工作表）
- **📥 导入数据**：导入Excel文件，支持：
  - 创建新事件（自动读取事件信息）
  - 导入到现有事件（自动检测重复）
  - 重复数据处理：跳过 / 覆盖 / 都保留
  - **无需密码**：简化流程，直接导入
- **智能预览**：导入前预览数据，确认无误再导入
- **多设备同步**：设备A导出 → 发送文件 → 设备B导入
- **即时反馈**：导入成功显示统计信息

### 🎙️ 智能语音播报

- **自动播报**：录入礼金时自动语音播报确认信息
  - 示例：`张三，贰仟元整，现金`
  - 示例：`李四，壹仟元整，微信，备注：新婚快乐`
- **操作反馈**：修改、删除、成功、失败都有语音提示
- **状态显示**：表单底部显示语音就绪状态
- **零依赖**：使用浏览器原生语音合成，无需额外安装

### 🔒 数据安全

- **本地存储**：数据只存在你电脑里，不会上传任何服务器
- **明文存储**：数据以JSON格式存储，方便导出和备份
- **无需密码**：简化操作，无需记忆密码
- **零上传**：绝不泄露隐私

### 📦 单文件应用

- **只有一个 HTML 文件**：下载即用，无需安装
- **离线可用**：支持 file:// 协议，双击打开
- **轻量级**：~320KB，快速加载

**数据导入导出：**

## 💡 小贴士

- 💾 **定期备份**：务必定期导出备份，防止浏览器数据丢失
- 🎙️ **语音播报**：无需看屏幕，听声音确认录入，适合忙碌场合
- 🖥️ **副屏功能**：连接投影仪展示给来宾查看
- 📄 **打印存档**：活动结束后打印保存珍藏
- 🔄 **多设备同步**：导出Excel → 发送文件 → 导入Excel，轻松同步
- 📊 **Excel统一**：导入导出都用Excel，简单方便

## ❓ 常见问题

**Q: 需要安装软件吗？**
A: 不需要！只有一个 HTML 文件，双击就能用。

**Q: 数据安全吗？**
A: 非常安全。数据存储在本地浏览器，不会上传任何服务器。

**Q: 数据会丢失吗？**
A: 浏览器清除数据会丢失。务必使用"导出备份"功能定期备份！

**Q: 副屏如何同步？**
A: 主屏录入后，副屏2秒内自动更新并滚动到最新记录。

**Q: 语音播报怎么用？**
A: 录入礼金后会自动播报，无需设置。支持Chrome、Edge、Firefox、Safari。

**Q: 语音播报不工作怎么办？**
A: 1) 检查浏览器是否支持；2) 首次使用可能需要点击允许语音权限；3) 确保浏览器音量开启

**Q: 会收费吗？**
A: 永久免费，开源项目。
