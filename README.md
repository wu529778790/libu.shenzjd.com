## ✨ 核心特性

### 🖥️ 副屏展示（投影专用）

- **实时同步**：主屏录入，2秒内自动更新
- **大字体显示**：标题和数据字体放大，远距离清晰可见
- **单行展示**：显示最新的12条记录，无空白区域
- **最新高亮**：最后一条记录脉冲动画高亮
- **高度自适应**：完美填充屏幕，视觉舒适
- **主题适配**：喜事红色 / 丧事灰色

### 📄 打印存档

- **横屏A4**：充分利用纸张空间
- **全量打印**：一次性打印所有礼金记录
- **彩色主题**：喜事红色 / 丧事灰色自动适配
- **竖排格式**：传统礼簿风格，符合中式习惯
- **自动统计**：包含总金额、人数、支付方式统计

### 💾 数据备份

- **一键导出**：下载 JSON 备份文件
- **Excel导出**：导出礼金明细 + 统计汇总表格
- **智能导入**：自动合并数据，跳过重复记录
- **多设备同步**：设备A导出 → 发送文件 → 设备B导入
- **即时反馈**：导入成功显示统计信息

### 🔒 数据安全

- **本地存储**：数据只存在你电脑里，不会上传任何服务器
- **AES-256加密**：没密码谁也打不开
- **零上传**：绝不泄露隐私

### 📦 单文件应用

- **只有一个 HTML 文件**：下载即用，无需安装
- **离线可用**：支持 file:// 协议，双击打开
- **轻量级**：~320KB，快速加载

## 📖 快速开始

### 首次使用

1. 下载 `index.html`
2. 双击打开
3. 选择：✨ 创建新事件 或 📂 导入备份

### 日常操作

**录入礼金：**

1. 点击"录入礼金"
2. 输入姓名和金额
3. 自动显示中文大写金额
4. 保存即可

**开启副屏（投影）：**

1. 主界面点击"开启副屏"
2. 自动打开最大化窗口
3. 连接投影仪或副屏展示

**打印礼簿：**

1. 主界面点击"打印/PDF"
2. 自动打开新窗口（横屏A4）
3. 自动调用打印对话框

**数据备份：**

- 💾 导出备份：保存为 .json 文件
- 📊 导出Excel：导出礼金明细 + 统计汇总（带中文大写金额）
- 📂 导入备份：选择 .json 文件自动合并

## 💡 小贴士

- 🔑 **记住密码**：忘记密码无法找回数据
- 💾 **定期备份**：务必定期导出备份，防止浏览器数据丢失
- 🖥️ **副屏功能**：连接投影仪展示给来宾查看
- 📄 **打印存档**：活动结束后打印保存珍藏
- 🔄 **多设备同步**：导出 → 发送 → 导入，轻松同步

## ❓ 常见问题

**Q: 需要安装软件吗？**
A: 不需要！只有一个 HTML 文件，双击就能用。

**Q: 数据安全吗？**
A: 非常安全。数据加密后存储在本地浏览器，不会上传任何服务器。

**Q: 数据会丢失吗？**
A: 浏览器清除数据会丢失。务必使用"导出备份"功能定期备份！

**Q: 副屏如何同步？**
A: 主屏录入后，副屏2秒内自动更新并滚动到最新记录。

**Q: 会收费吗？**
A: 永久免费，开源项目。
